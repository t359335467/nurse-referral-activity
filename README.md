# 护士网拉新领资料/小课活动 · 原型

## 说明
护士网拉新裂变活动原型。用户进入活动页浏览资料/小课，点击感兴趣的资料发起"邀请2位新用户助力"任务，分享给好友后助力成功即可领取资料。

## 页面清单
| 页面 | 说明 |
|------|------|
| 需求总体说明 | 整体业务流程说明（全宽独立页面） |
| 活动主页 | 推荐轮播 + 全部资料列表，含状态角标 |
| 裂变发起 | 资料详情 + 邀请进度 + 分享 |
| 受邀助力 | 受邀用户助力 + 登录引导 |
| 助力成功 | 新用户助力成功反馈 |
| 老用户提示 | 老用户无助力资格，引导成为发起人 |
| 已领取提示 | 已完成资料的去APP查看提示 |

## 布局规范
- 一级菜单切换页面
- "需求总体说明"为全宽独立文档页
- 其他页面左右分栏（左侧375px手机原型 + 右侧说明文档）
- 最大宽度1920px，全屏友好
